First Time Installation

Before this extension can be used, you must change the OTRS path option below.

Some websites may not behave as expected with this extension, and a whitelist
is provided which will allow you to conditionally enable this extension on a
per-site basis, if needed.


Linkify OTRS Issues: Options




Note: You may enter partially domain names and use regular expressions.
If the whitelist is empty, the extension is applied to all websites. If the whitelist contains data, the
extension is applied only to addresses included in the whitelist.


Advanced Settings
default: \b[\d]{16}\b
= a numeric string with exactly 16 digits. I.e. '2016040773000073'.
